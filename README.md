# Data_Anreg
Data Variabel Penelitian Faktor-Faktor yang Memengaruhi PDRB per Kapita Menurut Provinsi (2024)
Y   = Produk Domestik Regional Bruto per Kapita Atas Dasar Harga Berlaku
X1  = Indeks Pembangunan Manusia
X2  = Angkatan Kerja
X3  = Produktivitas Tenaga Kerja
X4  = Kepadatan Penduduk
X5  = Tingkat Pengangguran Terbuka
